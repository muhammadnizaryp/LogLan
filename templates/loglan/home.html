{ Log-Lan }

Log-Lan (Logarithm-Language) adalah sebuah e-learning bahasa pemrograman python berbasis Gamification, menyediakan fitur level belajar, materi pemrograman interaktif, kuis pilihan ganda, tantangan un-lock materi dan kuis, poin user untuk meningkatkan rangking.

Mulai Belajar

Materi diberikan dalam bentuk teks tertulis, contoh program, dan user diwajibkan untuk menyelesaikan tantangan coding program di setiap materi yang diberikan, tantangan tersebut dapat diselesaikan pada text editor yang telah disediakan.
