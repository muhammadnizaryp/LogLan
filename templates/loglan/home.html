Selamat Datang

{ Log-Lan }

Log-Lan (Logarithm-Language) adalah sebuah e-learning bahasa pemrograman Python berbasis Gamification, menyediakan fitur level belajar, materi pemrograman interaktif, kuis pilihan ganda, tantangan un-lock materi dan kuis, poin user untuk meningkatkan rangking.

Mulai Belajar

Materi belajar diberikan dalam bentuk teks tertulis, contoh program, dan user diwajibkan untuk menyelesaikan tantangan coding program di setiap materi yang diberikan, tantangan tersebut dapat diselesaikan pada text editor yang telah disediakan.

Kuis diberikan dalam bentuk pilihan ganda. Kuis berfungsi untuk mengasah pemahaman user terhadap materi yang telah diberikan disetiap level pembelajaran. Kuis dapat dikerjakan jika user telah menyelesaikan semua materi belajar pada level terkait.

Lock-Unlock materi belajar dan kuis berfungsi untuk menambah tantangan dalam proses belajar.

User dapat memperoleh poin dari setiap berhasil menyelesaikan koding program dan mengerjakan kuis

Hasil proses belajar user ditampilkan dalam bentuk Rangking, sehingga user mengetahui pada peringkat berapa proses belajar yang telah dilakukan. Rangking diambilkan dari total poin yang dikumpulkan user

Progres belajar user disajikan dalam bentuk presentase progres bar. Pada setiap level pembelajaran, user memiliki progres bar belajar masing-masing.
